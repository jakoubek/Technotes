# SQL Anywhere: Arbeit mit Tabellen






## Tabelle umbenennen

```sql
ALTER TABLE alter_tabellenname RENAME neuer_tabellenname;
```


## Tabelle löschen

```sql
DROP TABLE tabellenname;
```


## Defaultwert einer Tabellenspalte ändern

```sql
ALTER TABLE tabellenname MODIFY spalte1 DEFAULT 1;
```
